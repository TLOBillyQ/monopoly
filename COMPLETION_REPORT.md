## 架构亮点

### 1. 反应式状态管理
```lua
local money = State.Create(100000)
-- 变化自动通知所有依赖
money:Set(50000)
```

### 2. 自动计算属性
```lua
local totalAsset = Memo.new("TotalAsset", function(s)
    local m = s:D(money)
    local p = s:D(propertyValue)
    return m + p
end, {money, propertyValue})
```

### 3. 副作用处理
```lua
local logger = Effect.new("MoneyLogger", function(s)
    local m = s:D(money)
    print("金币: " .. m)
end, {money})
```

### 4. 事件系统
```lua
local onBankrupt = Trigger.Create("onBankrupt")
onBankrupt:Fire({playerId = 1})
```

## 与旧版本的对比

| 特性 | 旧版本 | 新版本 |
|-----|-------|-------|
| 状态管理 | 全局变量/table | Spoke State |
| 属性计算 | 手动计算 | Memo自动缓存 |
| 事件处理 | 回调函数 | Trigger + Effect |
| 代码组织 | 单一文件 | 模块化系统 |
| 可维护性 | 低 | 高 |
| 可扩展性 | 低 | 高 |
| 性能 | 普通 | 优化 |
| 代码量 | ~1500行 | ~2000行（含框架） |

## 性能优化

1. **Memo缓存** - 避免重复计算
2. **依赖追踪** - 只更新需要的部分
3. **对象池** - SpokePool支持内存优化
4. **批量更新** - 支持事务式更新
