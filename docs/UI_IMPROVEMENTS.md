## UI系统架构

### 核心组件

1. **RenderSystem (渲染系统)**
   - 负责游戏主画面渲染
   - 绘制游戏板、玩家、信息面板
   - 实现了渲染管道模式

2. **UI (UI组件系统)**
   - 提供可重用的UI组件
   - 对话框、地块卡片、按钮等
   - 独立的事件处理

3. **AnimationSystem (动画系统)**
   - 管理所有游戏动画
   - 骰子动画、移动动画
   - 基于时间的平滑过渡

### UI布局

## 技术亮点

1. **响应式设计**：使用Spoke框架的State管理，UI自动响应游戏状态变化
2. **模块化架构**：UI组件独立可复用，易于维护和扩展
3. **平滑动画**：基于时间的动画系统，确保流畅的视觉效果
4. **交互友好**：鼠标悬停、点击反馈，提升用户体验
5. **信息清晰**：使用颜色编码和图标，信息一目了然
