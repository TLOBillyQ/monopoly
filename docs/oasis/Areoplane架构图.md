## 回合状态详细流程

```
┌─────────────────────────────────────────────────────────┐
│                 单个回合状态转换                          │
└─────────────────────────────────────────────────────────┘

状态1: WaitForRollDice (等待掷骰子)
├─ 显示: 掷骰子按钮亮起
├─ 倒计时: 15秒（可配置）
└─ 动作: 等待玩家点击或超时自动掷

              │ (点击/超时)
              ▼

状态2: DiceRolling (骰子滚动)
├─ 显示: 骰子旋转动画
├─ 持续: 2秒（固定）
└─ 服务器: 计算随机结果 (1-6)

              │ (动画结束)
              ▼

状态3: WaitForPlaneSelection (等待选择飞机)
├─ 显示: 高亮可移动的飞机
├─ 倒计时: 15秒（可配置）
├─ 逻辑: 
│   ├─ 如果没有飞机可移动 → 自动结束回合
│   ├─ 如果只有一架飞机可移动 → 自动选择
│   └─ 多架飞机可选 → 等待玩家选择
└─ 动作: 等待玩家点击或超时自动选择

              │ (选择完成)
              ▼

状态4: PlaneFlying (飞机飞行)
├─ 显示: 飞机移动动画
├─ 持续: 根据移动格数计算
├─ 特殊事件:
│   ├─ 落在对手位置 → 触发吃子事件
│   ├─ 到达终点 → 触发到达事件
│   └─ 所有飞机到家 → 触发完成事件
└─ 音效: 播放飞行音效

              │ (动画结束)
              ▼

状态5: RoundEnd (回合结束)
├─ 检查1: 是否掷到6点?
│   └─ Yes → 不切换玩家，再掷一次
├─ 检查2: 玩家是否所有飞机到家?
│   └─ Yes → 记录完成时间
├─ 检查3: 是否3位玩家完成?
│   └─ Yes → 进入游戏结算
└─ 默认: 切换到下一位玩家

              │
              ▼
         (下一回合开始)
```

   ▼

   │
