## 移动规则（向前）

- **外圈默认方向**：外圈“向前”按逆时针方向移动。
- **移动方向（facing）**：每次移动都沿“当前方向”前进；若未指定，则沿上一回合记录的方向继续（例如“向下/向右”）。
- **进入内圈（入口点）**：入口点为 `45 / 40 / 44 / 43`（地图四边中点）。当且仅当玩家是沿外圈逆时针方向抵达入口点时：
	- 本次点数为**偶数**：进入内圈；进入后不回头，继续向前。
	- 本次点数为**奇数**：不进入，继续沿外圈逆时针前进。
- **黑市 39 分叉**：当移动路径会“越过黑市39”（即走到 39 后还要继续走下一步）时：
	- 本次点数为**奇数**：在 39 处按当前方向**左转**进入对应支路。
	- 本次点数为**偶数**：在 39 处按当前方向**右转**进入对应支路。

### 例子

- 例子1：起点=海口路`3`，步数=4，终点=天津路`32`。
- 例子2：起点=天津路`32`，当前移动方向向下（下一格为`31`），步数=6，终点=澳门路`6`。
- 例子3：起点=南昌路`25`，当前移动方向向右，步数=12，终点=南宁路`7`。

补充：入口点位于地图四边中点；若持续使用偶数步数并按规则进入内圈，则整体会呈逆时针绕圈效果。
