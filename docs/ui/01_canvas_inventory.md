## 层级建议

- loading_screen 在最上层（覆盖全部）。
- base_screen 常驻底层。
- market_panel 与 modal_choice、modal_popup 覆盖 base_screen。
- overlay_mask 可放在 base_screen 或弹窗画布，按需提高遮罩层级。
