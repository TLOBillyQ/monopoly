## 🎮 游戏流程

```
1. 初始化游戏
   ├─ 加载配置 (config.lua)
   ├─ 创建玩家 (player.lua)
   ├─ 初始化地块 (property.lua)
   ├─ 加载机会卡 (chance.lua)
   └─ 加载道具 (item.lua)

2. 游戏主循环
   ├─ 回合开始 (BEFORE_ACTION)
   ├─ 投掷骰子 (ROLL_DICE)
   ├─ 移动玩家 (MOVE)
   ├─ 着陆事件 (LAND_EVENT)
   │  ├─ 普通地块：购买/升级/租金
   │  ├─ 特殊地块：对应事件
   │  └─ 可能触发机会卡/道具
   └─ 回合结束 (AFTER_ACTION)

3. 游戏结束
   └─ 检测破产和最后的获胜者
```
