# 蛋仔大富翁（无 Spoke 版）

本分支彻底移除了 Spoke 依赖，重写为纯 Lua 的轻量实现。核心是一套简单的状态机驱动：掷骰 → 移动 → 结算 → 结束回合，支持自动模式与最基本的地产/机会卡/道具逻辑。

## 快速开始

```bash
love .          # 运行游戏
lua test_suite.lua   # 运行小型自检
```

按键说明：`SPACE` 下一步 · `A` 切换自动/手动 · `B` 买地 · `U` 升级 · `S` 跳过当前提示 · `H` 帮助 · `ESC` 退出。

## 模块结构

- `GameManager.lua`：游戏主状态机，处理回合/掷骰/结算/自动模式。
- `config.lua`：简化版配置（16 格棋盘、机会卡、道具、配色等）。
- `player.lua`、`property.lua`：玩家与地块操作。
- `chance.lua`、`item.lua`：精简的机会卡与道具逻辑。
- `render.lua`、`input.lua`：LÖVE2D 渲染与输入处理。

## 本版特性

- 5x5 边框路径的 16 格棋盘，起点、医院、税务局、黑市、机会/道具格均已保留。
- 道具为即时生效型（免费卡、加倍骰子、幸运护符、急救包），无需额外 UI 操作。
- 机会卡覆盖金钱波动、移动、传送、收付金币、抽取道具。
- 破产时自动释放名下地块。
