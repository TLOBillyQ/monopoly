# Monopoly Game

This context defines the shared game language for the Eggy Monopoly project. It keeps UI labels and turn-flow terms aligned with player-facing game semantics.

## Language

**可选行动阶段**:
当前玩家可以选择是否继续使用可用道具或其他可选效果的阶段。玩家完成该阶段后，回合继续进入后续必经流程。
_Avoid_: 行动, 任意操作阶段

**行动按钮**:
当前玩家从回合开始的行动等待进入必经回合流程的按钮。它不用于完成可选行动阶段；当结束按钮可用时，行动按钮不应同时作为可点击的推进入口。
_Avoid_: 继续按钮, 泛用推进按钮

**结束按钮**:
当前玩家在可选行动阶段表示不再继续使用可选行动、让回合流程继续的按钮。它使用自带“结束”字样的按钮图片，不额外写入“结束”文字；它只面向当前玩家，不显示给非当前玩家。可选行动阶段一开始即可点击，不需要二次确认。它不是强制结束整个回合，也不跳过掷骰、移动、落地或回合收尾；倒计时到期等价于结束按钮；它不用于扣留、空可选阶段、托管行动、弹层活动或动画等待。
_Avoid_: 结束回合按钮, 跳过回合按钮, 完成按钮, 跳过按钮, 返回按钮

**道具获得展示**:
玩家在对局中实时成功获得新道具卡时，用于先看清新卡的短暂放大展示；初始化、存档恢复、测试档案等非实时获得不触发。偷窃来源的获得者是偷窃者，不是被偷窃目标。展示可由玩家提前关闭，未关闭时 3 秒后自动结束；展示结束后的“收入卡槽”是视觉归位语义，不表示道具获得在规则上被延迟。
_Avoid_: 入包延迟, 获得延迟

**道具获得展示队列**:
多张道具卡在短时间内连续获得时，按照获得顺序逐张播放的道具获得展示。队列播放期间后续回合推进和相关交互等待展示完成；玩家提前关闭当前展示时，队列继续进入下一张或恢复后续流程。
_Avoid_: 合并展示, 只展示最后一张

**导弹目标玩家**:
使用导弹卡时由玩家选择的其他玩家。导弹效果轰炸该目标玩家当前所在格；格内建筑、障碍和同格未免疫玩家按导弹效果结算。
_Avoid_: 任意目标地块, 目标建筑
