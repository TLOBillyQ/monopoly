## 命名规则（建议）

为避免重名与便于程序按名称查找，统一采用「类型前缀 + 语义」：

- `Canvas_`：`ECanvas`
- `Img_`：`EImage`（含容器、遮罩、图标、卡槽、骰子等）
- `Lbl_`：`ELabel`
- `Btn_`：`EButton`

格式建议：`<前缀><模块>_<语义>_<可选序号>`

- 例：`Btn_HUD_Roll`、`Lbl_Popup_Title`、`Img_Shop_Item_01`
- 序号固定两位：`_01/_02/...`

## 通用层级（建议）

所有画布内部尽量遵循 4 层（均用 `EImage` 做容器）：

1. `Img_*_BG`：背景/底图
2. `Img_*_HUD`：常驻信息与操作区
3. `Img_*_Mask`：半透明遮罩（弹窗/黑市/偷窃选择时启用，阻断底层点击）
4. `Img_*_Panel`：弹窗/商店主体面板

## 通用组件（复用命名）

- Toast（提示）：`Img_*_Toast` + `Lbl_*_ToastText`
- 倒计时：`Img_*_Timer` + `Lbl_*_TimerText`（策划：单步最大等待 10 秒，超时默认确认）
