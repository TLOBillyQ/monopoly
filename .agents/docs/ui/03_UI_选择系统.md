# UI：选择系统（V2）

## 当前状态与目标状态

- 当前状态：选择交互仍偏向旧“单一通用选择屏”模型。
- 目标状态：按语义分流到四类选择画布：玩家、位置、遥控骰子、建筑升级。

## 路由规则（总览）

按 `choice.kind` 决定目标画布，优先级从上到下：

1. `market_buy` -> `黑市屏`
2. `remote_dice_value` -> `遥控骰子屏`
3. `item_target_player` -> `玩家选择屏`
4. `roadblock_target` / `demolish_target` -> `位置选择屏`
5. `landing_optional_effect` / `land_optional_effect` 且选项全集属于 `{buy_land, upgrade_land}` -> `建筑升级屏`
6. 其他 kind -> `位置选择屏`（通用 7 槽复用）

> 说明：第 5 条是本次新增的建筑升级分支。

## A. 玩家选择屏

画布：`玩家选择屏`

节点：

- `玩家选择_标题`
- `玩家选择_副标题`
- `玩家选择_槽位1`
- `玩家选择_槽位2`
- `玩家选择_槽位3`
- `玩家选择_确认按钮`
- `取消按钮`

规则：

- 选项按原顺序映射到 `槽位1..3`，超出 3 个截断。
- 点击槽位只改变“当前选中项”，提交由 `玩家选择_确认按钮` 触发。
- `取消按钮` 触发 `choice_cancel`。

主要 kind：

- `item_target_player`

## B. 位置选择屏（目标类 + 通用类）

画布：`位置选择屏`

节点：

- `位置_副标题`
- `位置_放置文本`
- `位置前1`、`位置前2`、`位置前3`
- `位置后1`、`位置后2`、`位置后3`
- `位置脚下`
- `位置_放置确认`
- `取消按钮`

规则：

- 静态容量 7：前 3 + 后 3 + 脚下 1。
- 超出 7 个选项按原顺序截断。
- `位置脚下` 仅在选项显式表示“脚下/当前位置”时显示；否则隐藏。
- 点击候选按钮只更新选中项；`位置_放置确认` 执行 `choice_select`。
- `取消按钮` 执行 `choice_cancel`。

明确归入本屏的 kind：

- `roadblock_target`
- `demolish_target`
- `item_phase_choice`
- `discard_item`
- `steal_item`
- `steal_prompt`
- `rent_card_prompt`
- `tax_card_prompt`
- `market_vehicle_replace`
- 其他未识别 kind（兜底）

关于 `landing_optional_effect` / `land_optional_effect`：

- 若 options 全是 `buy_land`/`upgrade_land`，走 `建筑升级屏`；
- 否则走本屏。

## C. 遥控骰子屏

画布：`遥控骰子屏`

节点：

- `遥控骰子_标题`
- `遥控骰子_正文`
- `遥控骰子_选项_01` ~ `遥控骰子_选项_06`
- `遥控骰子_取消`

规则：

- 6 个点数按钮直接触发 `choice_select`。
- `遥控骰子_取消` 触发 `choice_cancel`。

主要 kind：

- `remote_dice_value`

## D. 建筑升级屏（本次新增重点）

画布：`建筑升级屏`

节点：

- `建筑升级_标题`
- `建筑升级_确定按钮`
- `建筑升级_取消`
- （可选视觉节点）`建筑升级_标题背景`、`建筑升级_弹窗背景`

### 触发条件

仅当同时满足：

1. `choice.kind` 为 `landing_optional_effect` 或 `land_optional_effect`；
2. `choice.options` 的 `option_id` 全属于 `{ buy_land, upgrade_land }`。

### 文案规则

- 若当前目标操作是 `buy_land`，标题显示“购买地块”。
- 若当前目标操作是 `upgrade_land`，标题显示“加盖建筑”。
- 本屏不依赖旧通用正文节点。

### 确认/取消语义

- `建筑升级_确定按钮` -> `choice_select`
- `建筑升级_取消` -> `choice_cancel`

### 选项规则

- 当前玩法下，`buy_land` 与 `upgrade_land` 在同一落点互斥，通常只有 1 个候选。
- 若未来出现非地产选项混入，定义为回退到“通用路由”（即位置选择屏）。

## 按钮语义防冲突清单

- `取消按钮` 在机会卡屏 = 确认。
- `取消按钮` 在玩家/位置选择屏 = 取消。
- `建筑升级_确定按钮` 仅用于建筑升级确认。
- `关闭` 仅用于黑市取消。
