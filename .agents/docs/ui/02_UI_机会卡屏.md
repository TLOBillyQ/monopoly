# UI：机会卡屏

画布：`机会卡屏`

## 节点

| 节点 | 类型 | 用途 |
|---|---|---|
| `机会卡_标题` | ELabel | 标题 |
| `请输入文字` | ELabel | 正文 |
| `机会卡_图片` | EImage | 卡图/插图 |
| `取消按钮` | EButton | 确认/继续 |

## 展示规则

`push_popup(payload)` 时：

1. 切到 `机会卡屏`。
2. 标题写入 `机会卡_标题`。
3. 正文写入 `请输入文字`。
4. 若 `payload` 含图片引用，更新并显示 `机会卡_图片`；否则重置为空图并隐藏。
5. 点击 `取消按钮` 触发 `popup_confirm`，关闭弹窗。

## 返回规则

- 弹窗可中断当前模态屏。
- 关闭后优先恢复被中断模态屏，无法恢复时回 `基础屏`。

## 语义约束

- 本屏 `取消按钮` 的语义是“确认/继续”。
- 黑市使用 `关闭`，建筑升级使用 `建筑升级_取消`，不复用本屏语义。
