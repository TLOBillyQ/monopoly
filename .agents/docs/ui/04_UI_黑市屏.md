# UI：黑市屏

画布：`黑市屏`

## 节点

### 控制

- `黑市购买按钮`：确认购买
- `关闭`：取消/退出

### 预览

- `售价：100`：显示 `售价：{price} {currency}`
- `选中卡牌`：显示当前商品图

### 槽位

- `黑市购买项1..10`
- `道具名称1..10`
- `底框1..10`

## 交互规则

1. 打开黑市时，按 options 顺序渲染前 10 项。
2. 默认选中首个可选商品并刷新价格与预览。
3. 点击 `黑市购买项{i}` 只更新选中项（`market_select`）。
4. 点击 `黑市购买按钮` 提交当前选中项（`market_confirm -> choice_select`）。
5. 点击 `关闭` 触发 `choice_cancel`。

## 边界

- 黑市与其他模态屏互斥。
- 黑市失败提示由 `卡牌展示屏` 承接。
- `关闭` 只用于黑市，不参与其他画布的取消语义。
