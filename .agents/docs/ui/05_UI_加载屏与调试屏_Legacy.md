# UI：加载屏与调试屏

## 加载屏

画布名：`加载屏`。

### 节点

| 节点名 | 类型 | 用途 |
|--------|------|------|
| `加载提示` | ELabel | 加载提示文字 |

游戏初始化期间切换到此画布，显示加载信息。加载完成后切回基础屏。

## 调试屏

画布名：`调试屏`。

### 节点

| 节点名 | 类型 | 用途 |
|--------|------|------|
| `日志` | ELabel | 日志文字 |

### 行为

- 不参与画布互斥，独立叠加。
- 通过 `ui_view.set_debug_visible(state, true/false)` 控制可见性。
- 通过 `ui_view.set_debug_log(state, text)` 更新日志内容。
- `_switch_canvas` 切换时保留调试屏的可见状态（`debug_visible == true` 时不隐藏）。
