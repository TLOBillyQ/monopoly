# UI：地块与卡牌弹窗（购买/加盖/强征/免租/免税/破产）

对应策划：走到地块根据所有者弹不同弹窗；金币不足提示；可选使用强征卡/免费卡/免税卡；破产弹窗。

## Root

- `Canvas_Popup`（`ECanvas`）

## 层级结构（命名 / 类型）

```text
Canvas_Popup (ECanvas)
└─ Img_Popup_Root (EImage)
   ├─ Img_Popup_Mask (EImage)                     # 压暗主场景
   └─ Img_Popup_Panel (EImage)
      ├─ Lbl_Popup_Title (ELabel)                 # 地块名 / “破产” / “提示”
      ├─ Img_Popup_Content (EImage)
      │  ├─ Lbl_Popup_Line1 (ELabel)              # 价格/租金/税额/总价
      │  ├─ Lbl_Popup_Line2 (ELabel)              # 备用：加盖后租金/所有者等
      │  └─ Lbl_Popup_Desc (ELabel)               # 提示词（如“你有强征卡，是否使用？”）
      ├─ Btn_Popup_Left (EButton)                 # 放弃/算了
      └─ Btn_Popup_Right (EButton)                # 购买/加盖/使用/确定
```

## 场景文案与按钮（建议）

- 空地购买：
  - `Title=地块名`
  - `Line1=空地价格 xxxx`
  - `Line2=空地租金 xxxx`
  - `Right=购买`，`Left=放弃`
- 自己地块加盖（未到高楼）：
  - `Line1=本次加盖价格 xxxx`
  - `Line2=加盖后租金 xxxx`
  - `Right=加盖`，`Left=放弃`
- 他人地块强征提示（有强征卡）：
  - `Line1=地块所有者 xxx`
  - `Line2=地块当前总价 xxxx`
  - `Desc=你有强征卡，是否使用？`
  - `Right=使用`，`Left=放弃`
- 他人地块免租提示（有免费卡）：
  - `Line1=地块所有者 xxx`
  - `Line2=地块当前租金 xxxx`
  - `Desc=你有免费卡，是否使用？`
  - `Right=使用`，`Left=放弃`
- 税务局免税提示（有免税卡）：
  - `Desc=你本次需要支付所得税 xxxx，你有一张免税卡，是否使用？`
  - `Right=使用`，`Left=放弃`
- 破产：
  - `Title=破产`
  - `Desc=很遗憾，你破产了！`
  - 仅显示 `Btn_Popup_Right=确定`（隐藏左按钮）
