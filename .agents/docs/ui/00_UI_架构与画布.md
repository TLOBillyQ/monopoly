## 画布清单与职责

| 画布名 | 类型 | 职责 |
|---|---|---|
| `基础屏` | 常驻 | HUD、玩家信息、行动按钮、道具槽位 |
| `玩家选择屏` | 模态 | 玩家目标类选择 |
| `位置选择屏` | 模态 | 位置目标类 + 通用类选择 |
| `遥控骰子屏` | 模态 | `remote_dice_value` 点数选择 |
| `机会卡屏` | 模态 | 所有 `push_popup` 弹窗 |
| `建筑升级屏` | 模态 | 地产可选操作（购买/升级）确认 |
| `黑市屏` | 模态 | `market_buy` 专属购买界面 |
| `加载屏` | 过程态 | 启动加载提示 |
| `调试屏` | 叠加层 | 日志显示，不参与模态互斥 |
| `破产展示屏` | 保留 | 当前轮次不纳入选择系统 |

## 互斥与叠加规则

- `基础屏` 始终作为底层。
- 模态组同一时刻最多显示一个：
  - `玩家选择屏`
  - `位置选择屏`
  - `遥控骰子屏`
  - `机会卡屏`
  - `建筑升级屏`
  - `黑市屏`
- `加载屏` 只在初始化阶段短时覆盖。
- `调试屏` 独立叠加，不参与互斥。

## 选择路由总表

| 路由条件 | 目标画布 |
|---|---|
| `choice.kind == "market_buy"` | `黑市屏` |
| `choice.kind == "remote_dice_value"` | `遥控骰子屏` |
| `choice.kind == "item_target_player"` | `玩家选择屏` |
| `choice.kind in {"roadblock_target", "demolish_target"}` | `位置选择屏` |
| `choice.kind in {"landing_optional_effect", "land_optional_effect"}` 且 `option_id` 全在 `{buy_land, upgrade_land}` | `建筑升级屏` |
| 其他 choice kind | `位置选择屏`（通用 7 槽复用） |

## 容量与静态绑定约束

| 画布 | 静态按钮容量 | 规则 |
|---|---:|---|
| `玩家选择屏` | 3 | `玩家选择_槽位1..3`，超出截断 |
| `位置选择屏` | 7 | `位置前1..3` + `位置后1..3` + `位置脚下`，超出截断 |
| `遥控骰子屏` | 6 | `遥控骰子_选项_01..06` |
| `黑市屏` | 10 | `黑市购买项1..10` |
| `建筑升级屏` | 1（当前玩法） | 地产可选当前是单选确认流 |

## 按钮语义基线

| 场景 | 确认按钮 | 取消/关闭按钮 |
|---|---|---|
| `机会卡屏` | `取消按钮`（语义为“确认/继续”） | 无独立取消 |
| `玩家选择屏` | `玩家选择_确认按钮` | `取消按钮` |
| `位置选择屏` | `位置_放置确认` | `取消按钮` |
| `遥控骰子屏` | 直接点选 `遥控骰子_选项_*` | `遥控骰子_取消` |
| `建筑升级屏` | `建筑升级_确定按钮` | `建筑升级_取消` |
| `黑市屏` | `黑市购买按钮` | `关闭` |
