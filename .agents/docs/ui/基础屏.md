## 运行时行为（2026-02）

- `基础_行动日志按钮`：唯一的调试屏开关入口。点击后切换 `调试屏` 显隐。
- `基础_行动提示`：仅在回合 `start` 阶段触发时显示给当前回合玩家，作为“轮到你”的开场提示。
- `基础_其他玩家行动提示`：仅显示在非行动玩家客户端，文本将把模板中的“玩家”替换为动作发起者的客户端显示名。
