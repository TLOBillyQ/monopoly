# 本项目
- 本项目使用的lua环境说明见 `docs/lua_env.md`与`docs/lua_advanced.md`。
- 运行代码在 `src/`，核心规则在 `src/core/` 与 `src/gameplay/`；入口 `main.lua`、`src/game.lua`。
- Eggy API 检索：先看 `docs/eggy/api/`，再按关键词查 `docs/eggy/EggyAPI.lua`，在EggyAPI.lua中五次检索没命中，视为没有。
- 常用命令：`export_xlsx.bat`、`lua tests/deps_check.lua`、`lua tests/regression.lua`。
- 关于蛋仔编辑器的最新文档知识，见https://u5-creator.s3.game.163.com/manual/pc_md/overview.html
